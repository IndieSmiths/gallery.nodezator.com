How to add a node pack here?
To have your node pack added to this node gallery/database just submit the relevant info about your node pack in this form. Alternatively, you can also submit a pull request to the gallery.nodezator.com repository on github following the instructions on the README.md file. You can also email me (kennedy@kennedyrichard.com) if you need any help.
